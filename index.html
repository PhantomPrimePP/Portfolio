1v1 Simulator (Prototype)

This game is inspired by Aimlabs. While Aimlabs focuses on solo aim training, I thought, why not make it competitive? So, I added a second player to enable 1v1 battles with a friend. This game is a prototype and is still in development, as it is my first multiplayer game built using Unity and Mirror.

Ruins of Arachnids

This project focuses on game design and serves as a prototype scene developed using Unreal Engine 5. The game follows an Indian soldier who, during battle, falls into ancient ruins filled with giant arachnid creatures. Left alone after losing all his comrades, he must navigate the ruins and escape using ancient gadgets he discovers along the way.
